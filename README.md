# Delta AI

A revolutionary AI helper for learning anything.

This software is completely free and open source. This application relies on bringing your own key (BYOK) to function. A lightweight model is suggested since no reasoning is required.

We respect your privacy. All your data stay on your machine except for the AI requests. A built-in model will be used to decide the necessary information needed to generate content, and only a summarized version of your learning preferences will be sent.

# Requirements

- Use AI to quickly explain terms under cursor
	- Can possibly be used specifically as dictionaries
- Explain terms in AI explanations infinitely and recursively
- Maintain a smart knowledge base that grow with the user
- Use existing knowledge in the knowledge base in explanations
- Psychologically trustworthy
