# Delta AI

A revolutionary AI helper for learning anything.

This software is built using electron-vite + typescript + react.

This software is completely free and open source.

This application relies on bringing your own key (BYOK) to function. A lightweight model is suggested since no reasoning is required.

We respect your privacy. All your data stay on your machine except for the AI requests. A built-in model will be used to decide the necessary information needed to generate content, and only a summarized version of your learning preferences will be sent.

# Core Idea

- ### Goal
  - The primary goal is to help life-long learners approach brand new concepts theories, whether they are general, technical or cultural.
  - The tool will help them approach new things with lower difficulty and make learning a smooth and approachable process.
  - The tool is fully open source with MIT license. It relies on BYOK to function.
- ### Look-Up Feature
  - Use lightweight AI to quickly explain terms quickly under cursor using OCR.
  - Expand explanation windows and look up terms inside infinitely and recursively.
  - Use existing knowledge about the user to optimize explanations.
- ### Knowledge Base Feature
  - Maintain a smart, growing knowledge base for the user, including the user's learning preferences, current knowledge range, and personalities.
  - The users are able to see and manage their AI models and their knowledge bases.
  - The details of everything about the users are selected and summarized before sending to cloud (if it is) providers to get personalized results for the look-up.
- ### Look and Feel
  - Simplicity are preferred to ensure the users adopt the tool itself smoothly.
  - The colors and shapes will be simplistic and psychologically peaceful.
  - They will give the users a feeling of safety.
